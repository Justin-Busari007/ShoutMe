1. Overview

ShoutMe is a location-based event discovery platform that allows users to:

Discover nearby events

Filter events by category

View event details on a map

Create an account

Save and manage events

Interact with other users (if messaging enabled)

The platform works in a web browser and is optimized for desktop use.


2. Creating an Account

Navigate to the Register page.

Enter:

Full Name

Email Address


Password

Click Register.


3. Logging In

Go to the Login page.

Enter your email and password.

Click Login.

You will be redirected to the Home page.

If your credentials are incorrect, an error message will appear.


4. Home Page

The Home page allows you to:

View events on an interactive map

Browse event listings

Filter events by category

Search for events


Map Features

Click on a marker to view event details.

Zoom in/out using the map controls.

Events are displayed based on their geographic location.


5. Filtering by Category

Locate the Category Filter section.

Select a category (e.g., Music, Sports, Comedy, Art).

The page will update automatically to show only matching events.


6. Viewing Event Details

To view full details:

Click on an event card or map marker.

You will see:

Event Title

Description

Date and Time

Location

Category


7. Creating an Event

Navigate to Create Event.

Fill in:

Event Title

Description

Category

Date & Time

Location

Submit the form.

The event will appear on the map and event list.
