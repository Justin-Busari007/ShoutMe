---

- **1. Overview**
  - ShoutMe is a location-based event discovery platform designed to help users find and share events within their area.
  - Users can:
    - Discover nearby events using an interactive map
    - Filter events by category
    - View detailed event information
    - Create an account to access full features
    - Create and manage their own events
  - The platform operates through a web browser and is optimized for desktop use.

---

- **2. Creating an Account**
  - Navigate to the **Register** page.
  - Enter the required details:
    - Full Name
    - Email Address
    - Password
  - Click **Register** to create your account.
  - Once registration is successful, you may proceed to log in.

---

- **3. Logging In**
  - Go to the **Login** page.
  - Enter your registered email address and password.
  - Click **Login**.
  - Upon successful authentication, you will be redirected to the Home page.
  - If incorrect credentials are entered, an error message will be displayed.

---

- **4. Home Page**
  - The Home page serves as the main dashboard of the platform.
  - From here, users can:
    - View events displayed on an interactive map
    - Browse event listings
    - Filter events by category
    - Search for specific events

  - **Map Features**
    - Click on a map marker to view event details.
    - Use the zoom controls to adjust the map view.
    - Events are positioned according to their geographic location.

---

- **5. Filtering by Category**
  - Locate the **Category Filter** section on the Home page.
  - Select a category (e.g., Music, Sports, Comedy, Art).
  - The displayed events will automatically update to reflect the selected category.
  - To view all events again, clear or reset the filter.

---

- **6. Viewing Event Details**
  - To access full event information:
    - Click on an event card or corresponding map marker.
  - The event detail view includes:
    - Event Title
    - Description
    - Date and Time
    - Location
    - Category

---

- **7. Creating an Event**
  - Navigate to the **Create Event** page.
  - Complete the event creation form by providing:
    - Event Title
    - Description
    - Category
    - Date and Time
    - Location
  - Review the entered information for accuracy.
  - Click **Submit** to publish the event.
  - Once successfully created, the event will be visible on both the interactive map and the event listing section.
